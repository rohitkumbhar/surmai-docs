# Manage Traveler Profiles

## What are Traveler Profiles?

Traveler Profiles allow you to store and manage essential information for yourself and your fellow travelers in one
central location. Instead of re-entering details for every new trip, you can create a profile once and easily add it to
any trip you plan.

## Key Features

- **Centralized Information**: Store passport details, loyalty program numbers, and other essential travel information.
- **Document Attachments**: Upload and store copies of important documents like visas, vaccination certificates, or
  identification for quick access during your travels.
- **Reusable Profiles**: Create a profile once and use it across multiple trips, ensuring information is consistent and
  easy to find.

## Profile Ownership and Access

- **Updating Your Own Profile**: To update your personal travel profile, navigate to the **User Profile** page (
  accessible via your profile icon in the top-right corner) and go to the **Travel Profile** tab.
- **Managing Other Profiles**: All other traveler profiles you have access to can be managed on the **Travelers** page,
  which can be found in the sidebar.
- **Email-Based Profiles**: Each traveler profile is uniquely identified by an email address.
- **Ownership Transfer**: If you create a profile for someone using their email, you initially own that profile.
  However, as soon as that person creates their own Surmai account with the same email, ownership of the profile is
  automatically transferred to them.
- **Collaborative Management**: You can share a profile with other users by adding them as **Profile Managers**. This
  allows them to view and edit the profile information, which is particularly useful for families or groups traveling
  together.

## Adding Profiles to Trips

Once a traveler profile is created, it can be added to any trip. This ensures that all relevant information and
documents for that traveler are immediately accessible to anyone with access to the trip details.

::: warning Privacy and Data Security
Traveler profile data can contain highly personal information. Please note that this information is visible to all site
administrators. Exercise caution and follow your organization's security guidelines when adding sensitive details to a
profile.
:::
